# How to use it

### Get Group/Room user profile 

![](images/profile.jpg)

1. Type any word in group/room.
2. It will show group/room user profile.

### Leave Group/Room

![](images/leave.jpg)

1. Type "bye".
2. Chatbot will leave a group/room.
